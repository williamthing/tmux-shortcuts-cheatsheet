# tmux-shortcuts-cheatsheet

Managing tmux sessions:

Starting a new tmux session

			$ tmux

List running sessions on a terminal

			$ tmux ls

Opens previous tmux session and deletes all other tmux instance that are active

			$ tmux a -d

split panel
